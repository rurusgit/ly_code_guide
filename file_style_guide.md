[2 目录命名](#目录命名) 

[3 文件命名](#文件命名) 


## 1 项目命名 

全部采用小写方式， 以下划线分隔。

示例：
```
my_project_name
```

## 2 目录命名 
参照项目命名规则； 
有复数结构时，要采用复数命名法。 
示例： 
```
scripts, styles, images, data_models
```

## 3 文件命名 
全部采用小写方式， 以下划线分隔。
示例: 
```
account_model.js
```
```
retina_sprites.css
retina_sprites.scss
```
```
error_report.html
```
